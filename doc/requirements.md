# typeNow 需求清单

## 功能需求

### P0 — 核心功能

- [ ] **监听输入法切换**：实时捕获 macOS 系统输入法切换事件
- [ ] **获取当前输入法名称**：正确获取中/英/其他输入法的本地化名称
- [ ] **浮层弹出提示**：切换发生时，屏幕中央显示半透明浮层
- [ ] **自动消失**：浮层显示 1~2 秒后自动淡出消失
- [ ] **不抢焦点**：浮层出现不影响当前正在输入的应用

### P1 — 体验增强

- [ ] **淡入淡出动画**：浮层出现和消失有平滑过渡
- [ ] **状态栏图标**：可选显示状态栏小图标，便于退出和设置
- [ ] **登录自启**：提供"开机启动"选项
- [ ] **显示位置记忆**：用户可拖动浮层，记住位置

### P2 — 配置项

- [ ] **显示时长调节**：0.5s / 1s / 2s / 3s 可调
- [ ] **显示位置选择**：屏幕中央 / 光标附近 / 自定义
- [ ] **浮层大小调节**：小 / 中 / 大
- [ ] **透明度调节**：30% ~ 90%
- [ ] **暗色/亮色模式跟随**：自动适配系统外观

### P3 — 扩展（未来）

- [ ] **多语言输入法图标**：中文显示"中"，英文显示"EN"，日文显示"あ"
- [ ] **自定义提示文字/图标**
- [ ] **声音提示**：可选的音效反馈
- [ ] **触控栏支持**：在 Touch Bar 上显示

## 非功能需求

- [ ] 内存占用 < 30MB（常驻后台）
- [ ] CPU 占用 < 1%（空闲时）
- [ ] 支持 macOS 13+ (Ventura)
- [ ] 支持 Intel 和 Apple Silicon 芯片
- [ ] 应用包体积 < 5MB

## 不做的事

- 不支持 Windows / Linux
- 不修改系统输入法设置
- 不拦截或改变输入法切换行为
- 不上架 Mac App Store（初期作为独立 .app 分发）
